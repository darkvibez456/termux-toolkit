# Contributing to Termux Toolkit

Thanks for your interest in contributing! 🙌

---

## How to Contribute

### 1. Fork the Repo
Click the **Fork** button on GitHub.

### 2. Add Your Tool
Follow this format in the correct category in `README.md`:

```
| ![ToolName](https://img.shields.io/badge/ToolName-COLOR?style=flat-square&logoColor=white) | Short description | `install command` |
```

### 3. Rules
- Tool must work on Termux / Android
- Must be free and open source
- No duplicate tools
- Short and clear description in English

### 4. Submit Pull Request
- Title: `Add [ToolName] to [Category]`
- Describe what the tool does
